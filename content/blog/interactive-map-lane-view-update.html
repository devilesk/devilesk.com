--- title: Interactive Map Lane View Update description: > Interactive dota 2 map updated lane view. created: !!timestamp '2015-05-25 11:00:00' tags: - Interactive Dota Map - update post_images: images/blog/interactive-map-lane-view-update/ --- {% mark image -%} ![Interactive Map Lane View Update]([[!!images/blog/interactive-map-lane-view-update/header.jpg]]) {%- endmark %} {% mark excerpt -%} I've added a lane view feature to the [dota 2 interactive map](/dota2/apps/interactivemap2) that lets you view the map from a normal player perspective. {%- endmark %} To enter lane view mode just select Lane View option in the map controls box. Then click the part of the map you want to view. ![Map Controls]([[!!images/blog/interactive-map-lane-view-update/JhLh3D8.png]]) Lane view also shows the neutral camp spawn boxes. ![Neutral camp spawn boxes]([[!!images/blog/interactive-map-lane-view-update/yuu8wGf.jpg]]) Views are also linkable.
